    ErrorEntrega

    Devuelve true en el caso de que se haya producido algún problema en la entrega, habitualmente:

    • Por problemas de red local que impiden hacer el request
    • El servicio/end-point está caído
    • Errores en la validación del esquema (provocan un SOAPFault)
    • Otros problemas de red o cortes que impiden que se complete la operación

    En estos casos o no se ha obtenido una respuesta completa del servicio o rechaza la entrega por un SOAPFault.
