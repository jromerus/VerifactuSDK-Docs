    Class VerifactuManager

    Clase "Helper" que agrupa las funcionalidades necesarias para la generación de los mensajes xml-soap, la comunicación con el servicio para el registro de facturas y la gestión de certificados.

    AltaRegistro(RegFactuSistemaFacturacion, String, String)

    Realiza el alta del registro de facturación. Si no se indica un certificado exportado se usará el último certificado establecido (ver EstableceCertificado). Si no se ha esteblecido un certificado ni se ha indicado una ruta, el sistema mostrará un diálogo de selección de certificado (sólo en plataforma windows).
