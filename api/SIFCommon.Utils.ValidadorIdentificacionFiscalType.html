    Class ValidadorIdentificacionFiscalType

    Clase para la validación de NIF de personas físicas y jurídicas. Invoca a la clase estática ValidadorIdentificacionFiscal sin aportar más funcionalidad, pero resulta necesaria cuando la librería se usa desde sistemas no .NET mediante interop/COM ya que no pueden invocar clases estáticas (precisan instanciar)
