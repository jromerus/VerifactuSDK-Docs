    Documentación de Verifactu SDK.

    Bienvenido a la documentación técnica de las clases que conforman el API SDK de Verifactu

    Notas rápidas:

    1. El API al completo se encuentra en la sección Documentación API.

    2. Para comprender el SDK conviene destacar 4 clases principales alrededor de las cales "pivota" todo el API:

      1. RegFactuSistemaFacturacion: Esta clase ayuda modelar el documento RegFactuSistemaFacturacion que representa un lote de remisión de facturas, estipulado en la normativa siguiendo la definición de los esquemas xsd para la posterior generación del documento xml de intercambio.
      2. RegistroFacturaType: Esta clase representa un registro de facturación que puede ser tanto un registro de alta como de anulación.
      3. VerifactuManager: Esta clase de tipo "helper" presenta una serie de métodos para ayudarnos con la gestión de los certificados, la el alta de los registros y la obtención de los resultados de las operaciones para su posterior tratamiento.
      4. InfoQR: Con caracter previo a la generación de los registros de comunicación a Verifactu, en el propio proceso de emisión de la factura debemos generar un código QR que deberá aparecer en el documento de factura. Esta clase nos ayudará a generar tanto el código como la propia imagen representativa del mismo.

    3. Como complemento al SDK se ofrece una documentación técnica que explica tanto los mecanismos de generación de los registros de facturación, las distintas operaciones y los mecanismos y estrategias de control de flujos.
